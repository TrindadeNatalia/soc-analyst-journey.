  SOC Analyst Journey – Natália Trindade
Hoje iniciei oficialmente minha transição para a área de Cibersegurança com foco em SOC (Security Operations Center).

  O que é Cibersegurança?
    Cibersegurança é o conjunto de práticas, tecnologias e processos utilizados para proteger sistemas, redes e dados contra ataques digitais.

    Conceito CIA

Confidentiality (Confidencialidade) → Proteger informações contra acesso não autorizado.

Integrity (Integridade) → Garantir que os dados não sejam alterados indevidamente.

Availability (Disponibilidade) → Garantir que sistemas estejam acessíveis quando necessário.

Objetivo Profissional

Me tornar SOC Analyst Nível 1, com foco em monitoramento de alertas, análise de logs e resposta a incidentes.
